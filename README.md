# Giveaway Bot

A Telegram bot that manages giveaways, tracks referrals, and maintains a leaderboard. This bot uses the Pyrogram library for interaction with the Telegram API and MongoDB for data storage.

## Features

- **Start and Stop Giveaways**: The bot allows you to start and stop giveaways.
- **Registration**: Users can register for the giveaway and generate referral links.
- **Referral Tracking**: Track and manage user referrals.
- **Leaderboard**: Display the top 20 users based on the number of referrals.
- **Broadcast Messages**: Send messages to all participants.
- **Help Command**: Get a list of commands and their usage.

## Installation

### Prerequisites

- Python 3.7 or higher
- MongoDB instance (e.g., MongoDB Atlas)
